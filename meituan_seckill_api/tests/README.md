# 美团秒杀系统 API 测试

这个目录包含美团秒杀系统 API 的测试脚本。

## 测试脚本

主要测试脚本为 `test_api.py`，它测试所有 API 端点是否正常工作。测试结果使用Emoji图标展示，使结果一目了然。

## 运行测试

### 先决条件

1. 确保 API 服务器正在运行
2. 安装必要的依赖：

```bash
pip install requests
```

### 执行测试

```bash
# 从项目根目录运行
python tests/test_api.py
```

### 测试界面

测试脚本采用了友好的命令行界面，使用Emoji图标增强可读性：

- ✅ 表示测试通过
- ❌ 表示测试失败
- ℹ️ 表示信息提示
- ⚠️ 表示警告信息

### 测试范围

测试脚本覆盖以下功能:

1. **🔐 认证测试**
   - 登录
   - 获取当前用户信息

2. **👥 用户管理测试**
   - 获取用户列表
   - 创建新用户
   - 获取特定用户
   - 更新用户
   - 删除用户

3. **🔑 API密钥管理测试**
   - 获取API密钥列表
   - 创建新API密钥
   - 获取特定API密钥
   - 更新API密钥
   - 重置API密钥使用次数

4. **📋 抢券任务管理测试**
   - 获取任务列表
   - 创建新任务
   - 获取特定任务
   - 更新任务状态
   - 删除任务

5. **📊 访问日志测试**
   - 获取最近日志
   - 获取特定API密钥的日志
   - 获取时间范围内的日志

### 测试结果

测试完成后，脚本将输出测试统计信息，包括总测试数、成功数和成功率。结果以清晰的表格形式展示，结合Emoji图标使得成功和失败一目了然。
